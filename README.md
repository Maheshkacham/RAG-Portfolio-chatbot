# RAG Portfolio Chatbot

This is my first AI project. I built a chatbot that reads my resume and answers questions about me.


## About this project

I am a beginner learning Python and AI. I wanted to try building something with LangChain and Google Gemini, so I made a chatbot that can answer questions about my resume like:

- "What are Mahesh's skills?"
- "What is Mahesh's work experience?"
- "Does Mahesh know Python?"


## What I used

- Python
- LangChain
- FAISS (vector database)
- Google Gemini API
- PyPDF
- Google Colab

---

## How to run

1. Open `Portfolio.ipynb` in Google Colab
2. Upload your `resume.pdf` in the Colab Files sidebar
3. Get a free Google API key from https://aistudio.google.com/apikey
4. Add it as a Colab secret named `GOOGLE_API_KEY`
5. Run all cells one by one
6. Ask questions in the last cell


## What I learned from this

- What RAG (Retrieval-Augmented Generation) means
- How to load and split a PDF into chunks
- How vector databases store and search text
- How to connect LangChain with Google Gemini
- How to fix package conflicts in Google Colab
